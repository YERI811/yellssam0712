## 🧾 수업 개요

이 지도안은 **중학교 2학년 국어과 매체 읽기 단원**의 프로젝트형 수업 설계입니다.  
학생들이 **실제 데이터를 기반으로 문제를 발견하고**, **비판적으로 해석한 후**, **자신의 생각을 설득력 있게 표현**하는 과정을 경험하게 됩니다.  
모든 수업은 **파인만 기법(설명 중심 사고)**을 적용해 학생 친화적으로 설계되었습니다.
